Angular Final Project

Calculator

A calculator app that calculates add, subtract, multiply or divide result.

It uses two-way binding to retrieve input values and calculates the result when a button is pressed.

Text Editor

A text editor that allows users to make text bold, italic or underlined.

It uses two-way binding text area to apply formatting when button is pressed.

Weather

A weather app that allows users to search weather by city name.

It uses OpenWeather API to fetch and display weather data by user input.
